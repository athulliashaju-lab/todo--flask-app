TaskFlow - Flask To-Do Application

TaskFlow is a task management web application built using Flask and SQLite.

Features

- User Registration
- User Login & Logout
- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Completed
- Progress Tracking
- Search Tasks
- Dark Mode

Tech Stack

- Python
- Flask
- SQLite
- HTML
- CSS
- Bootstrap
